WhereTo.io

In this project, I developed an interactive web application, alongside my partner Fara, to help users locate Advice Drop-In centres in Bristol. It offers features such as finding centres on an interactive map, and listing information like services offered and addresses on a filterable table.

It is built using several web development technologies such as integrating external APIs, the Mapbox API to display an interactive map, allowing users to visualise the locations of the centres, and sourcing its data from Bristol Open Data. HTML and CSS is used for the front-end structure and styling of the web pages. Whilst JavaScript enables user interaction by accessing geolocation data and dynamically generating HTML content based on API responses.

For testing, I used QUnit to validate the functionality of the application. These tests focus on the accuracy the outputTable function, to ensure that it correctly processes and displays the data from the API. Specific checks include validating the number of rows generated in the table and ensuring that the data (such as venue name, address, and organisation) matches the expected output.

This project was not deployed as it was for a university project. You can still click on the link to my GitHub repo to discover the pages and further information about how we planned, designed and implemented the project.
