Hacker News Top Stories Scraper

My first individual project.

This is a Python script that scrapes top stories from the first and second page of Hacker News, and filters the results based on the number of votes. Only stories with 100 or more votes are included in the output and are sorted by the number of votes received.

Python Libraries:

I used the get function in the requests module to grab the first two pages of Hacker News. This is done live to when the user runs the code, and so the stories will constantly update based on what is currently on the website. Res.text is then used to retrieve the HTML string of the webpage.

The Python library, Beautiful Soup, is used to parse the HTML string. Specifically, Beautiful Soup is used to access the body tag and extract specific data from here (story titles, links, and vote counts). This makes it easier to manipulate as I work with only the relevant information I need to reach the objective of this project.

Finally, pprint is used to tidy up the output in a more presentable way for the reader to identify the story, the votes and click on the link directly to the story.
